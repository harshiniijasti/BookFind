
# 📚 BookFind

A modern book discovery platform that helps users search, explore,organize books  and analyze their progress


## 🚀 Live Demo

👉https://book-find-app-delta.vercel.app

A modern book discovery web application built with Next.js, React, TypeScript, Clerk authentication, and the Google Books API.


## ✨ Features

- 🔍 Search millions of books
- 📖 View detailed book information
- ❤️ Add books to Favorites
- 📚 Create and manage a Reading List
- 👤 Secure authentication with Clerk
- 📱 Fully responsive UI
- ⚡ Fast search powered by Google Books API
- 🎨 Modern and clean user interface

## 🛠️ Tech Stack

**Frontend**
- Next.js
- React
- TypeScript
- Tailwind CSS

**Authentication**
- Clerk

**API**
- Google Books API

**Deployment**
- Vercel
